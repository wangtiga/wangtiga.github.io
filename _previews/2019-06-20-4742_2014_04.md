## 5.真正衡量数字通信系统的有效性的指标是
### A.传输容量
### B.误码率
### C.输出信噪比
### D.频带利用率
> 答案 P42 1.8 通信基础 通信系统的性能指标

有效性，可靠性 是通信系统的主要性能指标。

衡量数字通信系统`有效性`的指标，传输速率 和 频带利用率：
- 1）码元速率$R_B$，波特率
  每秒传送的码元数目，
  单位Baud/s,
  详细内容参考1.5频谱和带宽->扩展知识。
- 2）信息速率$R_b$，比特率
  每秒传送的信息量，
  单位bit/s,
- 3）频带利用率，真正的衡量指标
  每赫兹(Hz)内所实现的传输速率，
  单位 Baud/Hz 或 bit/(s * Hz) ，
  $\eta = \frac{R_B}{B}$ 。

衡量数字通信系统`可靠性`的指标，差错率：
- 1)误码率
- 2)误信率

衡量模拟通信系统 有效性 的指标，传输带宽。

衡量模拟通信系统 可靠性 的指标，用接收端最终输出信噪比。

## 8.2ASK和2PSK所需的带宽是码元速率的
### A.2倍
### B.3倍
### C.4倍
### D.8倍
> 答案 P96 P122 3.4.1 调制和解调 二进制数字调制 二进制幅移键控

- 振幅键控(ASK Amplitude Shift Keying)
  有振幅为 1 ，没有振幅为 0 。
  没有振幅即，电压或电流一直保持不变。
- 频移键控(FSK Frequency Shift Keying)
  频率很低为 1 ，频率很高为 0 。
- 相移键控(PSK Phase Shift Keying)
  波形相对于 0 点的位置，就是相位。
  即，根据信号初始电压或电流的取值，区分 0 或 1 。

> 2ASK 的前缀 2 ，表示的是 二进制。即，我们只使用两种幅度表示两种不同的数据，满足二进制所需要的种类即可；如果要表达 八进制 ，则需要使用八种不同的幅度。

## 9.信宿是传递消息的目的地，其功能与信源
### A.相同
### B.相反
### C.相似
### D.相近

> 答案 P23 1.2.1 通信基础 通信系统的组成

```txt
    信源 ------> 发送设备 ------> 信道 -------> 接收设备 ------> 信宿
      |             |              ^                |             | 
      |             |              |                |             | 
      +----发送端---+              |                +----接收端---+ 
           			噪声源
```

- 信源
  消息发源地。
  通过传感器把消息转换为电信息,非电量－电量的转换。
  模拟信源，mic camera 。 
  数字信源，computer 。
- 发送设备
  匹配信源和信道。
  将信源产生的消息信号变换成适合信道传输的信号。
  电路，变换器：光端机，调制解调器（猫）,交换机。
  多路复用器：路由器，安防行业使用的矩阵。
- 信道
  有线信道：光纤，同轴电缆，双绞线。
  无线信道：大气，自由空间。
- 接收设备
  将原始电信号还原成消息。
  放大，反变换。滤波，译码，解调。
- 信宿
  消息目的地。
  将复原的原始电信号还原成相应的信息：扬声器。
